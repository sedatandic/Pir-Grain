**what currently exists?**
The application is a commodity trading dashboard with a feature-rich Market Data page. This page is composed of several tabs:
- **Indications (formerly Market News/Notes)**: A commentary section with a hierarchical date navigation (Year -> Month -> Weekday). Users can post and view comments for specific dates.
- **Live Prices**: A table displaying real-time scraped commodity and currency prices from Barchart.com, which auto-refreshes every 15 minutes.
- **Turkish Exchanges**: Displays scraped daily prices from both Konya Ticaret Borsası (KTB) and Gaziantep Ticaret Borsası (GTB).
- **TMO Tenders**: A module to create, view, and manage Turkish Grain Board tenders. The UI is a collapsible card layout per the user's design, with functionality to add, edit, and delete tender results.
- **Coaster Freights**: A new tab that scrapes weekly freight reports from . It features year/week navigation and displays the report PDF as a series of images alongside both the English and Russian text summaries.
- **Telegram Feed**: A right-hand sidebar displays a live feed of messages from 7 configured grain market Telegram channels, scraped from public web previews. Messages can be opened in an in-app popup.
